# ai-qa-workflow
Implementing Playwright MCP 

## Available Playwright MCP Tools

Use these tools directly during test execution. Do not write test code unless explicitly asked.

| Tool | Purpose |
|---|---|
| `browser_navigate` | Navigate to a URL |
| `browser_click` | Click an element by selector, text, or role |
| `browser_type` | Type text into an input field |
| `browser_fill` | Clear and fill an input field |
| `browser_select_option` | Select a dropdown option by value or label |
| `browser_check` | Check a checkbox |
| `browser_uncheck` | Uncheck a checkbox |
| `browser_screenshot` | Take a screenshot of the current page |
| `browser_snapshot` | Capture accessibility snapshot (DOM structure) |
| `browser_wait_for` | Wait for element, text, or URL condition |
| `browser_hover` | Hover over an element |
| `browser_press_key` | Press a keyboard key (Enter, Tab, Escape, etc.) |
| `browser_scroll` | Scroll the page or a specific element |
| `browser_evaluate` | Run JavaScript in the browser context |
| `browser_close` | Close the browser after completing the session |
| `browser_go_back` | Navigate back in browser history |
| `browser_go_forward` | Navigate forward in browser history |
| `browser_reload` | Reload the current page |
| `browser_handle_dialog` | Accept or dismiss browser dialogs or modals |
| `browser_resize` | Resize the browser viewport |
| `browser_network_requests` | Inspect network requests and responses |
